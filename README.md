# Sistema de Gerenciamento de Usuários

## Visão Geral

Este é um projeto da disciplina de Gerência e Controle de Mudanças da faculdade IFPB. Ele inclui uma interface para login e registro, além de funcionalidades para adicionar, remover, modificar e pesquisar usuários. O sistema é desenvolvido com HTML, CSS, JavaScript para o frontend, e Node.js com MySQL para o backend.

## Tecnologias Utilizadas

- **Frontend:**
  - HTML
  - CSS
  - JavaScript

- **Backend:**
  - Node.js
  - MySQL (banco de dados)
